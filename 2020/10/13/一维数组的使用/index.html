一维数组的使用

一维数组的声明和初始化

  1. ```java
    int[]arr1 = new int[]{1,2,3,4};
    
2. ```java
   int[]arr1 = new int[4];
  • 数组一旦初始化完成，其长度也就确定了

如何调用数组的指定位置的元素

int[] arr2 = new int[]{2,3,4,5};
System.out.println(arr2[2]);

其输出结果为4，数组的角标从0开始，到数组的长度-1结束

如何获取数组的长度

int[] arr3 = new int[]{2,3,4,5};
System.out.println(arr3.length);

如何遍历数组

int[] arr4 = new int[]{2,3,4,5};
for(int i = 0;i < arr4.length - 1;i++){
  System.out.println(arr4[i])
}

数组元素的默认初始化值

  • 数组元素为整型：0
  • 数组元素为浮点型：0.0
  • 数组元素为char型：0或’\u0000’，而非’0’
  • 数组元素为boolean型：false
  • 引用数据类型：null，即空值

数组的内存解析

  1. 先在栈中声明，然后在堆中开辟出一块连续的空间，进行默认初始化，通过开辟空间的首地址值与栈中的变量联系起来，当调用的时候，栈中变量通过首地址值进行调用
  2. 当变量出栈，指针消失，留存在堆中的空间变为无用空间，会在某个不确定的时间进行回收——垃圾回收机制

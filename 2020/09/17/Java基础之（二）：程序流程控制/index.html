Java基础之（二）：程序流程控制

本文最后更新于：2021年2月9日 凌晨

基本概述

概念

  • 流程控制语句是用来控制程序中语句执行顺序的语句，可以把语句合成能完成一定功能的小逻辑模块

分类

  • 顺序结构：程序从上到下执行，没有跳转和判断
  • 分支结构：根据条件选择性地执行某段代码
  • 循环结构：重复地执行某段代码

分支结构

if-else结构

三种格式一览

if(条件表达式){
  执行代码块
}
if(条件表达式){
  执行代码块1
}else{
  执行代码块2
}
if(条件表达式1){
  执行代码块1
}else if(条件表达式2){
  执行代码块2
}
……
else{
  执行代码块n
}

​

说明

  1. 实际开发中，else是可选的，可有可无
  2. 针对条件表达式：
    • 如果多个表达式之间是“互斥”关系，即无交集，则顺序无所谓‘
    • 如果多个表达式之间有交集，则需要根据实际情况，考虑清楚应该将哪个结构声明在上面
    • 如果多个表达式之间有包含关系，通常情况下，范围小的需要声明在范围大的上面，否则范围小的没机会被执行
    • if-else结构是可以嵌套使用的
    • if-else结构中的执行语句如果只有一行，对应的{}可以省略，但是不建议省略
    • else就近原则：于最近的if配对

switch-case结构

  1. 结构：

    • Switch(表达式){

      case 常量1:

      语句1;

      //break;

      case 常量2:

      语句2;

      //break;

      …………

      case 常量n:

      语句n;

      //break;

      default:

      语句;

      //break;

  2. 说明：

    • 根绝switch表达式中的值，依次匹配各个case中的常量，一旦匹配成功，则进入相应的case结构中，调用其执行语句，当调用完后，仍然会向下执行其他case结构中的执行语句，直到遇到break关键字或者到此switch-case结构末尾结束
    • break，可以使用在switch-case结构中，表示一旦执行到此关键字，就跳出switch-case结构
    • switch-case结构中的表达式只能是如下六种之一：byte,short,char,int,枚举类型（JDK5.0新增），String类型（JDK7.0新增）
    • case之后只能声明常量，不能声明范围
    • break关键字是可选的，根据实际开发情况选择加或者不加
    • default功能与else相当，也是可选的，而且位置相当灵活
    • 如果多个switch-case中的执行语句相同，可以考虑合并

if-else与switch-case的区别

  1. 凡是可以使用switch-case结构的，都可转换为if-else，反之不成立
  2. 二者均可的情况下，同时，switch表达式的取值情况不太多，优先使用switch-case

循环结构

概述

在某些条件满足的情况下，反复执行特定代码的功能

分类

  • for循环
  • while循环
  • do-while循环

循环语句的四个组成部分

  • 初始化部分（init_statement）
  • 循环条件部分（test_exp）
  • 循环体部分（body_statement）
  • 迭代部分（alter_statement）

for循环

  1. 格式

    for(1初始化条件;2循环条件;4迭代条件){
  3循环体
}

  2. 执行过程

    1-2-3-4-2-3-4-2-3-4……2

while循环

  1. 结构

    1初始化条件;
while(2循环条件){
	3循环体;
	4迭代条件;
}

  1. 说明

    • 写while循环不要忘记写迭代条件，一旦丢了，就可能导致死循环
    • for循环可以和while循环相互转换
    • 区别：for循环和while循环的初始化条件部分的调用范围不同

do-while循环

  1. 结构：

    1初始化条件;
do{
  3循环体;
  4迭代条件;
}while(2循环条件)

  2. 执行过程

    1-3-4-2-3-4-2……2

  3. 说明：do-while循环至少执行一次循环体

其他

最简单的死循环

  • while(true){

}

  • for(;;){

}

结束循环的方式

  1. 在循环部分返回false
  2. 在循环体部分执行break

嵌套循环

  1. 将一个循环结构A声明在另一个循环结构B的循环体中，就构成了嵌套循环
  2. 内层循环遍历一遍，相当于外层循环体执行了一次
  3. 外层循环控制行数，内层循环控制列数

特殊关键字的使用

  • break：在switch-case、循环结构中使用，结束当前循环
  • continue：在循环结构中使用，结束当次循环
  • 相同点：在关键字的后面不能声明执行语句
  • 如果想结束指定的某一层循环，则再次循环前加标签，然后在break后加标签，指明要结束的标签

Scanner类的使用：

  1. 作用：从键盘获取输入值

  2. 具体实现步骤：

    1. 导包：import java.util.Scanner;

    2. Scanner的实例化：Scanner scan = new Scanner(System.in);

    3. 调用Scanner类的相关方法，来获取指定类型的变量

  3. 注意：需要根据相应的方法，输入指定类型的值。如果输入的数据类型与要求的类型不匹配，会报异常：InputMissionException，导致程序终止

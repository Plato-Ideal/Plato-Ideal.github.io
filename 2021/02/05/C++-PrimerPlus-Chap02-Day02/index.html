2.4函数

函数(function)是C++的重要概念；它是一个程序所必需的部分；C++函数分为两种：有返回值的**和没有返回值的；

有返回值的函数

如函数sqrt()，它返回平方根；而函数返回的值可以赋给变量或者在其他表达式中使用；

x = sqrt(6.25); //return the value 2.5 and assigns it to the x

在这个例子中，sqrt()首先接收一个double类型的浮点数——6.25——然后返回一个double类型的浮点数；

在这个过程中，表达式sqrt(6.25)被称作：函数调用；而sqrt()这个被调用的函数被称为被调用函数(called function)；调用sqrt()的函数称为调用函数(calling function)；6.25为参数；sqrt(6.25)返回了一个值——2.5——这个值叫做返回值(return value)；

函数原型与函数定义

函数原型是一句告诉编译器函数的返回值及参数的语句，通常被放在程序的开头，被包含在头文件中；

函数定义则包含了函数代码，被包含在库文件中；

在程序中使用函数，必须提供函数原型，通常有两种方式实现：

  1. 在源代码文件中输入函数原型；

    double sqrt(double);

  2. 包含的头文件中定义了原型，在源代码中提供头文件；

    #include<cmath>

    函数变体

多个参数

有些函数并不是只有一个参数，而是可以输入多个参数，参数间用,分隔；

double y = pow(4,2); //表示4 * 4
   cout << y; //16

不接受参数

有些函数则不接受参数，可以用空括号或者void表示；

int x = rand(); //函数原型为 int rand(void);
   cout << x;

无返回值

有些函数接收参数却无返回值；

void simon(int n)
{
    cout << n;
}

无返回值的函数在调用时不可以放在赋值语句或其他表达式中；

用户定义函数

标准C库提供了140+的预定义函数，但有时这些函数却不能满足我们的需要，所以有时候我们需要自己设计函数；

要注意的是：如果自己设计函数，需要在main()定义之前提供函数原型，并在main()函数之后，提供其函数代码；

#include<iostream>
void simon(int); //函数原型 function prototype
using namespace std;

int main()
{
    simon(5);

    return 0;
}

void simon(int n) //函数定义
{
    cout << n;
}

其标准的函数格式应当如下：

return_value_type function_name(argumentlist)
{
	statements
}

需注意：不论是自定义函数还是其他函数，其地位都应当是一致的，即：函数定义不能嵌套在另一个函数定义中！

main()函数返回0的原因

由自定义函数我们可以知道这样一件事：int main()函数返回一个整数，并且不接收任何参数；所以我们可以看到在main()函数中有返回语句return 0;；那么我们不由发出疑问，main()函数为什么返回0？返回的0去了哪里？

首先，main()函数返回0值是一个约定俗称的事情，0代表无误，即程序在调用过程中没有出错，非0则表示程序在运行过程中出了问题；

其次，main()函数可以看作是操作系统与程序之间互动的接口，所以其返回值不是返回给程序的其他部分，而是返回给操作系统；

关键字

C++专用的字符；如，return、void、int等等；

特别注意，main不是关键字，但是，如果用其作为变量名会引发错误，所以最好不要这样做！！！！

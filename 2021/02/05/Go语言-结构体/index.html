Go语言-结构体

本文最后更新于：2021年2月9日 凌晨

结构体

Go语言中没有类的概念，也不支持类的继承等面向对象的概念，Go语言中通过结构体的内嵌再配合接口比面向对象具有更高的扩展性和灵活性。俗称面向接口编程

Go语言中有许多基本类型，如int、string、float32等等，可以用来描述一样事物的具体属性；但不足之处在于，基本属性不能够完整地描述一样事物的所有属性；所谓结构体，就是Go语言为了解决这个问题而创造的自定义类型；通过struct关键字，可以把全部属性都封装进结构体中；

结构体的定义

type book struct { //创建一个book结构体封装着关于book的属性
	name     string
	sendTime int
	price    float32
}

func main() {
	var dataStruct = book{
		name:     "Data Struct",
		sendTime: 17,
		price:    119.99,
	}
	fmt.Println(dataStruct)
}

除此以外，结构体也支持调用具体属性：

fmt.Println(dataStruct.name)
fmt.Println(dataStruct.sendTime)
fmt.Println(dataStruct.price)

结构体的实例化

第一种：

//第一种
var jinhong stu

//第二种

结构体的初始化

  1. 按照结构体创建的属性顺序写
type stu struct {
name   string
birth  int
height int
gender string
}
shaofeng := stu{
	"shaofeng",
	19990101,
	176,
	"男",
}
fmt.Println(shaofeng) //{shaofeng 19990101 176 男}

用这种方法，属性一定要写全，否则编译器会报错；

  1. 按照键值对方式初始化
type stu struct {
name   string
birth  int
height int
gender string
}
bilin := &stu{
	name:   "bilin",
	birth:  18190202,
	gender: "男",
}
fmt.Println(bilin)

此二者不能混用；

结构体内存布局

结构体中的成员变量在内存中是连续的；

bilin := &stu{
	name:   "bilin",
	birth:  18190202,
	gender: "男",
}
fmt.Println(&(bilin.birth))  //0xc0000b6370
fmt.Println(&(bilin.name))   //0xc0000b6360
fmt.Println(&(bilin.gender)) //0xc0000b6380

结构体为值类型

同基本类型、数组等值类型变量相同，结构体也为值类型；

构造函数

Go语言中没有构造函数的概念，但是我们可以自己实现：

type date struct {
	year  int
	month int
	day   int
}

func newDate(y, m, d int) date {
	return date{
		year:  y,
		month: m,
		day:   d,
	}
}

func main() {
	birth := newDate(1999, 12, 13)
	fmt.Println(birth) //{1999 12 13}
}

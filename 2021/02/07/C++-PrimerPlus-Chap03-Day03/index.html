Chap03 处理数据

3.1 简单变量

程序将信息存储在计算机中，主要记录三个属性：

  • 地址
  • 值
  • 类型

其过程为：程序找到一块可以存储的内存，然后用变量名(或其他标识符)标记，并将值赋给内存单元；然后就可以访问了；

变量名

C++命名规则：

  • 名称中只能使用字母、数字和下划线；
  • 名称的第一个字符不能是数字；
  • 区分大小写；
  • 不能用C++关键字；
  • 以两个下划线或下划线和大写字母打头的的名称将被保留给实现使用；以一个下划线打头的名称将被保留给实现使用，用作全局标识符；
  • C++对于名称长度没有限制；

整型

即没有小数部分的数字；C++中的基本类型有：

  • char
  • `short
  • int
  • long
  • long long(C++11新增)

其中，char最常用来表示字符，而非数字；

C++没有将整型的宽度定死，而是提供了一种更为灵活的机制：

  1. short至少16位；
  2. int至少和short一样长；
  3. long至少32位，且至少与int一样长；
  4. long long至少64位，且至少与long一样长；

通常来说，字节指的是8位内存单元，而C++中，字节由至少能够容纳其实现的基本字符集的相邻位组成，即可能取值的数目必须等于或超过字符数目；如在ASCII中，C++字节通常包含8位，而在Unicode中，有些实现可能使用16位甚至32位的字节；

sizeof

C++中内置了sizeof运算符用来返回类型或变量的长度，单位为字节；

//通过sizeof运算符，返回类型或变量的长度，单位为byte
#include<iostream>
#include<climits>
using namespace std;

int main()
{
	int n_int = INT_MAX;
	short n_short = SHRT_MAX;
	long n_long = LONG_MAX;
	long long n_llong = LLONG_MAX;

	cout << sizeof(int) << endl; //4
	cout << sizeof n_short << endl; //2
	cout << sizeof n_long << endl; //4
	cout << sizeof n_llong << endl; //8

	cout << n_int << endl; //2147483647
	cout << n_short << endl; //32767
	cout << n_long << endl; //2147483647
	cout << n_llong << endl; //9223372036854775807

	cout << CHAR_BIT << endl; //8

	return 0;
}

其中，INT_MAX表示int的最大取值，其被包含在头文件climits中；CHAT_BIT用来表示字节的位数；

头文件climits定义了符号常量来表示对类型的限制，其包含如下语句行：

#define INT_MAX 32767

#define与#include同为预处理器编译指令，其工作方式为：在程序中寻找INT_MAX，并将其替换为32767，程序在完成替换后被编译；预处理器寻找独立的标记，跳过嵌入的单词；

注意：在设计那些可用于C和C++的程序时，头文件必须使用#define；

初始化方式

初始化其实就是将变量赋值的过程；C++的初始化方式有很多，下面我们来逐一介绍：

  1. 标准赋值

    //1
int a0;
a0 = 13;
//2
int a = 123;

  2. 用已知值的变量去初始化

    int a = 123;
int b = a;

  3. 用已知值的表达式初始化

    int a = 123;
int b = a;
int c = a + b;

  4. 小括号初始化

    short d(322);

  5. 大括号赋值

    long e = { 1999 };
long f{ 1222 };
//也可以大括号里不写值，将初始化为0
long long g = {};
long long h{};

    大括号初始化用于数组和结构，在C++98中，也可以用于单值变量；之后在C++11中使用更为广泛；

无符号整型

上面介绍的short、int、long、long long都是有符号类型，即其值包含负数部分，而相应地也有无符号类型；运用unsigned关键字可以创建无符号类型变量；

unsigned int revert = 65536;
unsigned short uncles = 65535;

要注意的是，无论是有符号类型还是符号类型，一旦超过最大值或者低于最小值，它就会重置，超过最大值就会变成最小值，低于最小值就会变成最大值；

unsigned int revert = UINT_MAX; //设置为int的最大值
cout << revert << endl; //4294967295
revert += 1;
cout << revert << endl; //0
short zero = SHRT_MIN; //设置为short的最小值
cout << zero << endl; //-32768
zero -= 1;
cout << zero << endl; //32767

在如此多的整型之中，应当如何选择呢？通常，int被设计成自然长度，即计算机处理起来效率最高的长度，如无其他特殊情况，应当选用int；

进制表示

默认情况下，cout以十进制显示整数，无论其怎样书写：

int chest = 25; //十进制
int chest2 = 025; //八进制
int chest3 = 0x25; //十六进制

cout << chest << endl; //25
cout << chest2 << endl; //21
cout << chest3 << endl; //37

但是如果程序中确实要如实表示相应进制的数字，则应该使用控制符dec、hex、oct；由头文件iostream所提供；

int chest = 42;
int chest2 = 42;
int chest3 = 42;

cout << "十进制表示：" << chest << endl; //42
cout << "八进制表示：";
cout << oct;
cout << chest2 << endl; //52
cout << "十六进制表示：";
cout << hex;
cout << chest3 << endl; //2a

要注意的是：在修改格式之前，之前的格式一直有效，这也就是说，如果你将进制改为十六进制，之后的数字都会以十六进制呈现，除非你再次改回十进制；

常量存储

在C++中，在未指定的情况下，除非特殊情况，否则将存储为int类型；

如果要指定类型，则需要添加后缀；

  • u/U ——表示unsigned int
  • l/L —— 表示long
  • ul/UL——表示unsigned long
  • ll/LL —— 表示long long （C++11提供）
  • ull/ULL——表示unsigned long long

C++对于后缀的格式没有明确要求，顺序、大小写可以任意写；

char类型

char类型是专门为存储字符（如字母和数字）而设计的；现在更多用于存储字母；它可以表示目标计算机系统中的所有基本符号；char类型是一种特殊的整型；

char character;
cout << "Enter the character:";
cin >> character;
cout << endl << "It's your character:" << character << endl;

//Enter the character:M
//It's your character:M

在这个程序中，其工作过程为：

cin将键盘输入的M转换为77；输出时，cout将77转换为所显示的字符M；

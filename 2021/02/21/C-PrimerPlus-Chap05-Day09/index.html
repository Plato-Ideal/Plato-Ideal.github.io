5.2 while循环

while循环是没有初始化和更新部分的for循环，它只有测试条件和循环体；

其执行顺序与原理基本同for循环一致；

但要注意的是，因为while循环没有更新部分，所以要想最终停止循环，必须在循环体内添加一个可以影响测试条件值的部分；

cout << "Enter a word:";
char word[12];
cin >> word;
int i = 0;
while (word[i])
{
    cout << word[i] << ": " << int(word[i]) << endl;
    i++;
}

程序输出为：

Enter a word:Rossia
R: 82
o: 111
s: 115
s: 115
i: 105
a: 97

由于while循环没有初始化部分，所以我们需要先循环语句之前声明一个变量；

对于while循环的测试部分，因为C-风格字符串以'\0'作为结束的标记，而word[i]显示的是字符的值，当i推进到空字符时，其值将为0，循环将结束；这样写更为简洁，当然也更常用；还有一种较为复杂的写法：word[i] != 0，最终的结果是一致的；

另外，可以看到我们用i++;来影响while循环的测试条件部分；

for VS while

通常来说，用for循环可以编写的程序用while循环也可以，反之也成立；所以本质上来说，for循环与while循环等效；

但其二者有稍许不同：

  1. 在for循环中省略了测试条件时，将认为条件为true；
  2. 在for循环中，可以使用初始化语句声明一个局部变量，但在while中不可以；
  3. 如果循环体中包括continue语句，情况将稍有不同；

那么如何去判断何时该用哪个循环呢？

通常，程序员使用for循环来为循环计数，因为for循环将所有相关信息放在同一个地方；在无法预知循环将执行多少次时，程序员优先使用while循环；

延时循环

这里需要用到一个函数clock()，该函数被定义在头文件ctime中，用来返回程序开始执行后所用的系统时间；需要注意：clock返回的不一定是秒，其次该函数的返回类型不确定；

针对这个问题，头文件ctime提供了针对性解决方案：首先，它定义了一个符号常量——CLOCKS_PER_SEC,该常量等于每秒钟包含的系统时间数；因此，将其乘以秒数，就得到了以系统时间表示的该秒数；将系统时间除以CLOCKS_PER_SEC就得到了秒数表示的该系统时间；

其次，ctime定义了clock_t作为clock()返回类型的别名，然后由系统将其转换为其他类型（long或者unsigned int或其他适合系统的类型）；

由此我们编写一个根据用户输入来延迟的程序：

float secs;
cout << "Enter the delay time(s): ";
cin >> secs;
clock_t delay = secs * CLOCKS_PER_SEC;//将用户输入秒数转换为系统时间

cout << "Starting...\a\n";
clock_t  start = clock();//声明一个start变量表示程序开始

while (clock() - start < delay)
    ;
cout << "Done!\a\n";

该函数以系统时间为单位，避免了转换为秒所消耗的时间；

类型别名

类型别名是类型的另一个名字，本质上还是那个类型；

C++为定义类型别名提供两种方式：

  1. 使用预处理器；
  2. 使用C++关键字：tepedef；

首先来说第一种方法：

#define BYTE char

比较简单，之前我们也提过，这样编写代码将使得预处理器在执行代码之前将源文件中所有的char替换为BYTE，通过这样的方法，将BYTE作为char的类型别名；

然后就是通过关键字来定义类型别名：

typedef int alis;

在这条语句中，将使得alis作为int的类型别名；

与define相比，typedef是一种更好的选择，有时候也是唯一的选择；

5.3 do while循环

不同于for循环和while循环的入口条件循环，do while是一种出口条件（exit condition），即程序先执行循环体，然后再进行测试条件部分，这意味着do while循环至少执行一次；

通常来说，入口条件循环比出口条件循环要好，因为它在进入循环之前对条件进行检查，但在某些情况下，出口条件循环要更好一些；

#include <iostream>
using namespace std;

int main()
{
    cout << "Enter the number from 1 to 10:\n";
    cout << "Guess which number is my favorite.\n";

    int num;

    do
        {
            cin >> num;
        } while (num != 8);
    cout << "Yes,it's my favorite number.";

    return 0;
}

程序输出为：

Enter the number from 1 to 10:

Guess which number is my favorite.
4
6
8

Yes,it’s my favorite number.

显然在这种情况下，do while要优于for和while循环；

5.4 基于范围的for循环

在C++11中新增了一种循环——基于范围的for循环，它简化了针对于数组（或容器类——vector或array）的操作；

int i[5] {1,2,3,4,5};
for (int x:i)
{
    cout << x << endl;
}

程序遍历数组内每个元素并输出；

如果要修改数组内元素，则需要使用不同语法：

for (double x:i)
{
    x *= 0.8;
    cout << x << endl;
}

程序输出为：

0.8
1.6
2.4
3.2
4

5.5 循环和文本输入

在进行重复输入、逐字符获取文本输入时，使用循环是常见的；通常，使用while循环来完成这项任务；

cin对象只此3种不同模式的单字符输入，其用户接口各不相同；

cin输入

使用while循环执行此类任务时，我们需要知道程序何时停止，一般使用哨兵字符（sentinel character）——即选择某个特殊字符——来作为停止标记；

char ch;
int count = 0;

while (ch != '#')
{
    cout << ch;
    ++count;
    cin >> ch;
}

cout << "The sum of characters is " << count << endl;

上述程序将用户输入记录下来并统计字符数；

程序输出如下：

I love the sun and the moon.#But I do love you so much.
Ilovethesunandthemoon.The sum of characters is 23

但可以看到该程序并没有原样输出——它省略了空格；其原因在于cin;它在读取char值时，同读取其他类型一样，将忽略空格和换行符；只有当用户按下回车键时，输入的内容才会被发送给程序；

还有一个问题是，因为cin会暂时将当前输入缓冲，所以即使输入'#'，程序也没有及时停止，当用户按下回车键时，所有的字符序列被发送给程序，但程序在遇到哨兵字符时结束对剩余字符的处理；

所以我们可以在哨兵字符后面输入更多字符；

cin.get(ch)

使用cin对象的成员函数get()可以补救一部分cin的缺陷；

它将读取下一个字符——即使是空格——并将其赋给变量ch；

char ch;
int count = 0;

while (ch != '#')
{
    cout << ch;
    ++count;
    cin.get(ch);
}
cout << "The sum of characters are " << count << endl;

程序输出为：

I love the sun and the moon.#But I do love you so much.
I love the sun and the moon.The sum of characters are 29

可以看到，程序如实输出了空格；但输入依旧被缓冲，所以输入字符仍然比预定要多；

5.6 嵌套循环和二维数组

C++没有提供二维数组的类型，但用户可以创建每个元素都是数组的数组；

例如：int map[4][5];可以看作是一个包含4个元素，且每个元素都是包含5个元素的数组的数组；进一步地，可以将4看成行，5看作列；于是我们可以把二维数组看作一个表格；

如果要打印二维数组的每一个元素，需要使用for循环来遍历；

初始化二维数组

对二维数组初始化的方式同对结构数组的初始化方式基本雷同；都是要先初始化外层，然后再初始化里层；

int map[4][5]
{
    {1,2,3,4,5},
    {6,7,8,9,0},
    {11,13,14,43,65},
    {21,32,5,4,6}
};

=是可选的；

然后我们来遍历一下：

for (int row = 0; row < 4; row++)
{
    for (int col = 0; col < 5; col++)
    {
        cout << map[row][col] << "  ";
    }
    cout << endl;
}

程序输出为：

1 2 3 4 5
6 7 8 9 0
11 13 14 43 65
21 32 5 4 6

二维数组的使用

二维数组的使用本质上来说与一维数组的使用大同小异，都是可以按照索引去访问数组成员；

同时，二维数组也可以同指针数组相结合，例如：

//用来展示5年间4个城市的最大温度
#include<iostream>
using namespace std;

const int Cities = 4;//声明城市常量
const int Years = 5;//声明年份常量

int main()
{
    //声明一个指针数组，每一个元素都是一个指针，可被初始化一个字符串地址
    char * city[Cities]
    {
        "Gribble City",
        "Gribbletown",
        "New Gribble",
        "San Gribble"
    };

    //声明一个二维数组并初始化，记录4个城市5年间的最大温度情况
    int maxtemps[Cities][Years]
    {
            {97,96,98,97,99},
            {101,106,108,107,109},
            {87,86,88,87,89},
            {93,94,93,92,91},
    };

    cout << "Maximum temperatures for 2009 - 2012\n\n";
    for (int i = 0; i < Cities; i++)//用来输出城市
    {
        cout << city[i] << ":\t";
        for (int j = 0; j < Years; j++)//输出各年份温度
        {
            cout << maxtemps[i][j] << "\t";
        }
        cout << endl;
    }


}

程序输出为：

Maximum temperatures for 2009 - 2012

Gribble City: 97 96 98 97 99
Gribbletown: 101 106 108 107 109
New Gribble: 87 86 88 87 89
San Gribble: 93 94 93 92 91

改程序输出4个城市5年间的最大温度情况；可以看到它将一个char指针数组同一个int类型的二维数组相结合使用；

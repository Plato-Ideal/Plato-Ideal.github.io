hexo博客err:TypeError:cannot read the 'footnote' of undefined

本文最后更新于：2021年2月9日 晚上

erro:TypeError: cannot read the ‘footnote’ of undefined

错误如题；

一直想把标签(tags)页面删除，我觉得目前有一个分类(categories)已经足够，今天网上找了找相关的资料，发现大多数都是教怎么添加博客页面，很少有教怎么删除博客页面的；

于是自己动手，按照博客push的原理，就是把本地的资源上传到服务器存储，那么只要把本地的资料删除，网上自然也就不存在了；

找到博客根目录下的_config.yml，crtl+F找到menu选项，在tags前面加上#；再找到主题下面的根目录，不是thems的根目录，而是你具体主题的根目录，比如next主题等等；然后同样找到_confi.yml，同样找到menu选项，同样在tags前加#；

然后

hexo clean
hexo g
hexo d

结束！！

但是在操作过程中，却突然出现如题一般的错误；我再次查看了两个_config.yml文件，发现是#添加位置的问题

- { # key: 'tag', link: '/tags/', icon: 'iconfont icon-tags-fill' } //错误方式！！

正确方式应该是：

# - { key: 'tag', link: '/tags/', icon: 'iconfont icon-tags-fill' } //正确方式

修改完成，再次尝试，成功！！

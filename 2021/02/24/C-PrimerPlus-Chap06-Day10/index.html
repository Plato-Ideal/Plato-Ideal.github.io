Chap 06 分支语句与逻辑运算符

6.1 if语句

if语句是分支选择结构，用来帮助程序决定是否执行某个指令；

if语句有两种模式：if语句和if-else语句；

其中，if语句语法格式为：

if (test-condition)
    statement

测试条件部分通常为关系表达式；如同循环语句一样，测试条件将被强制转换为bool类型；即非零为true，零为false；

statement部分既可以是一条语句，也可以是语句块；

if-else语句语法格式为：

if (test-conditon)
    statement1
else 
    statement2

无论是if语句还是if-else语句，它们的整体都被视为一条语句；

除此之外，还有一个模式为：if-else-if-else语句，其实质是将一个if-else嵌套在另一个if-else中；用来帮助选择三个或三个以上的选项：

鉴于if-else结构比较好理解，所以这里用一个例子来展示一个具体用法，然后不再赘述：

/*此程序通过用户输入来猜数字*/

#include <iostream>
using namespace std;

const int num = 18;
int main()
{
    cout << "Enter a number to find my favorite(0-100):";
    int user_in;

    do {
        cin >> user_in;
        if (user_in < num)
        {
            cout << "Too low,guess again:";
        }
        else if (user_in > num)
        {
            cout << "Too high,guess again:";
        }
        else
        {
           cout << user_in << " is right.";
        }
    } while (user_in != num);

    return 0;
}

程序输出为：

Enter a number to find my favorite(0-100):50
Too high,guess again:25
Too high,guess again:12
Too low,guess again:16
Too low,guess again:17
Too low,guess again:18
18 is right.

6.2 逻辑表达式

总共分为三个：OR(||)、AND(&&)、NOT(!)；

其表示意义分别为：

  • ||——两侧表达式只要有一个为true则整个表达式的值为true；
  • &&——两侧表达式只要有一个为false则整个表达式为false；
  • !——对表达式取反操作，表达式为true，则取反后为false；

这三个逻辑运算符有如下性质：

  1. ||和&&被视为一个顺序点，即先修改左侧的值再对右侧的值进行判定；对于||来说，如果左侧表达式为true，则不必对右侧表达式进行判定，结果即为true；对于&&来说，如果左侧表达式为false，则将跳过对右侧表达式的判定，结果即为false；
  2. 运算优先级问题上，||和&&运算符的优先级都低于关系运算符；而!运算符的优先级高于所有的关系运算符和算术运算符；
  3. 为了防止某些计算机无法键入逻辑运算符的符号，C++将or、and和not定义为保留字，这意味着不能使用它们作为变量名等；

（PS：对于逻辑运算符这一块的内容比较好理解，运用起来也比较容易，为了节省篇幅及时间，我们不再赘述；）

6.3 字符函数库cctype

函数库cctype主要提供了一些确定字符是否为数字、字母大小写、标点符号等的函数；这些函数的原型被定义在头文件cctype中；其示例程序如下：

#include <iostream>
#include <cctype>

using namespace std;

int main()
{
    //统计用户输入的符号数，当用户键入'@'程序停止

    int whitspaces = 0;//空白
    int characters = 0;//字母
    int punct = 0;//标点符号
    int digits = 0;//数字
    int others = 0;//其他

    char user_in;//用户输入

    cin.get(user_in);

    while (user_in != '@')
    {
        if (isalpha(user_in))
        {
            characters++;
        }
        else if (isdigit(user_in))
        {
            digits++;
        }
        else if (isspace(user_in))
        {
            whitspaces++;
        }
        else if (ispunct(user_in))
        {
            punct++;
        }
        else
        {
            others++;
        }

        cin.get(user_in);//获取下次输入
    }

    cout << characters << " characters.\n"
        << whitspaces << " whitespace.\n"
        << punct << " puncts.\n"
        << digits << " digits.\n"
        << others << " others.\n";

    return 0;

}

best in 2027.”OK,It’s my favorite life.” I hope so.

142 characters.
45 whitespace.
13 puncts.
6 digits.
0 others.

6.4 条件运算符?:

?:条件运算符是C++中唯一一个需要三个操作数的运算符，常用来替代if-else；

其结构如下：

expression1?expression2:expression3

如果表达式1的结果为true则整个表达式的值为表达式2的值，否则为表达式3的值；

#include <iostream>
using namespace std;

int main()
{
    //输出最大值
    int a_num;
    int b_num;
    int max_num;

    cout << "Enter two num:\n";
    cin >> a_num >> b_num;

    max_num = (a_num > b_num) ? a_num : b_num;

    cout << "The max number is " << max_num << ".\n";

    return 0;
}

程序输出为：

Enter two num:
2 4
The max number is 4.

条件运算符与if-else不同的点是：

条件表达式生成一个表达式，因此是一个值，可以将其赋给变量或者放到一个更大的表达式中；

当代码复杂时，运用if-else可以表达的更清晰；

6.5 switch语句

switch语句在C++中作为一种分支语句，专门用来处理列表形式的选择结构；

其语法结构为：

switch (integer-express)
{
    case label 1:statement
    case label 2:statement
    ......
    default: statement
}

同if-else结构不同，switch语句中的括号内的表达式必须是一个结果为整数值的表达式；另外，每个标签都必须是整数常量表达式；常见的有int、char；当然，也可以是枚举量；

特别注意的是，在switch语句中，case标签并不是选项之间的界限，这就意味着，程序执行完特定标签并不会跳出switch语句块，而是按顺序执行到底，然后再跳出；如果哦想让程序执行完一组特点语句之后停止，必须使用break关键字；

#include <iostream>
using namespace std;
void showmenu();//函数原型
void report();
void comfort();

int main()
{
    showmenu();
    int choice;
    cin >> choice;
    while (choice != 5) 
    {
        switch (choice) 
        {
            case 1: cout << "\a\n";
                break;
            case 2: report();
                break;
            case 3: cout << "The boss was in all day.\n";
                break;
            case 4: comfort();
                break;
            default: cout << "That's not a choice.\n";
                break;
        }

        showmenu();
        cin >> choice;
    }
    cout << "Bye!\n";
    return 0;
}

void showmenu()
{

        "1) alarm\n"
        "2) report\n"

}

{
    cout << "It's a excellent week for business.\n"
        "Sales are up 120%.Expense are down 35%.\n";
}

{
    cout << "You're the finest CEO.\n";
}

switch与枚举

switch语句可以同枚举量进行结合，将枚举量用作标签；

#include <iostream>
using namespace std;

enum {red, blue, green, yellow};//生成0-3 四个数

int main()
{
    cout << "Enter a number:";
    int code;
    cin >> code;

    while (code >= red && code <= yellow)
    {
        switch (code) 
        {
            case red:cout << "It's red.\n";
            break;
            case blue:cout << "It's blue.\n";
            break;
            case green:cout << "It's green.\n";
            break;
            case yellow:cout << "It's yellow.\n";
        }
        cout << "Enter a number:";
        cin >> code;
    }
    cout << "Bye!\n";
    return 0;
}

程序输出为：

Enter a number:1
It’s blue.
Enter a number:2
It’s green.
Enter a number:3
It’s yellow.
Enter a number:4
Bye!

cin通常无法识别枚举类型;

当switch语句将int值和枚举标签进行比较时，将枚举量提升为int；在循环中，也会将枚举量提升为int类型；

switch与if-else

总的来说，if-else更适合处理取值范围，而如果选项可以用整数常量表示，那么所谓switch会是一个更好的选择；

6.6 break和continue

break和continue都可以用来停止循环，但是二者有不同的地方；

break是跳出整个循环，而continue是跳过此次循环；

6.7 读取数字的循环

在循环中获取数字并允许用户在数组填满之前结束输入或者在用户错误输入之后提示用户然后允许继续输入，如果要编写如上程序需要以下操作：

比如我们要编写一个程序来获取每天打鱼的重量，每天最多打5条鱼，然后计算平均重量，程序如下：

#include <iostream>
using namespace std;

const int Max = 5;

int main()
{
    cout << "Please enter a number,and the Max is " << Max << "(input 'q' to stop):\n";

    double fish[Max];
    cout << "fish #1: ";
    //获取数据
    int i = 0;
    while (i < Max && cin >> fish[i])
    {
        if (++i < Max)
        {
            cout << "fish #" << i+1 << ": ";
        }
    }
    //计算平均数
    double total = 0.0;
    for (int j = 0; j < i; j++)
    {
        total += fish[j];
    }
    //输出数据
    if (i == 0)
    {
        cout << "No fish\n";
    }
    else
    {
        cout << total / i << " = average weight of "
            << i << " fish\n";
    }
    cout << "Done.\n";
}

程序输出为：

fish #1:24
fish #2:34
fish #3:45
fish #4:33
fish #5:q
34 = average weight of 4 fish
Done.

可以看到我们用一个while循环来获取用户输入，而允许用户提前结束输入的操作为cin >> fish[i]；这个表达式实际上是一个cin方法调用，该函数返回cin；如果cin位于测试条件中，则将被转换为bool类型；如果输入成功，则转换后的值为true，否则为false；如果表达式为false则循环结束；

可以简单地理解为如果你输入的不是一个数字，则循环结束；

那么如果用户错误输入之后想继续输入怎么办？

这时，程序应该采取3个步骤：

  1. 重置cin以接受新的输入；
  2. 删除错误输入；
  3. 提升用户再输入；

示例程序如下：

#include <iostream>
using namespace std;

const int Max = 6;

int main()
{
    double fish[Max];

    cout << "Please enter the weight of fish:\n";
    //获取数据
    int i;
    for (i = 0; i < Max; i++)
    {
        cout << "fish #" << i+1 << ": ";
        while (!(cin >> fish[i]))
        {
            cin.clear();
            while (cin.get() != '\n')
            {
                continue;
            }
            cout << "Please enter a number: ";
        }
    }

    //计算平均数
    double total = 0.0;
    for (int j = 0; j < i; j++)
    {
        total += fish[j];
    }

    //输出数据
    cout << total / i << " = average of weight.\n";
    cout << "Done.\n";
}

程序输出为：

Please enter the weight of fish:

fish #1:25
fish #2:q
Please enter a number:23
fish #3:45
fish #4:67
fish #5:65
fish #6:must int?
Please enter a number:23
41.3333 = average of weight.
Done.

其中：

表达式!(cin >> fish[i])对输入进行判定，如果输入符合要求则cin表达式为true，因此将一个值放入到数组中；而如果输入不符合要求，则cin表达式为false，而!(cin >> fish[i])表达式为true，继而进入循环；

cin.clear();方法对输入进行重置，如果省略该语句，那么程序将不能继续进行输入；

cin.get()将获取直至行尾前的所有输入，然后通过continue跳过里层while循环，来达到删除错误输入的目的；

6.8 简单文件输入/输出

文本I/O是指程序读入文本文件或者输出到文本文件中；

在使用cin进行输入时，不管目标数据类型是什么，输入一开始都是字符数据——文本数据；然后cin对象负责将文本转换为其他类型；

例如：

38.5 19.2

假设上述数据是等待输入的输入行；根据目标数据类型的不同有如下不同情况：

  • 字符类型：则第一个字符3因为其输入类型与目标类型一致——均为字符类型——所以不需要转换；但需要知道，此刻存储在这里的不是数值3，而是字符3的编码；
  • int类型：此刻cin将不断读入，直到遇到非数字字符；也就是说，cin将读取3和8，句点被当做输入队列中的下一个字符；cin通过计算得到38，然后将38的二进制编码复制到变量中；
  • double类型：cin将不断读取，直到遇到第一个不属于浮点数的字符；在这里，cin读取3、8、句点以及5，然后将空格成为输入队列中的下一个字符；cin计算得到浮点数38.5，然后将其对应的二进制编码复制给变量；
  • char数组：在这种情况下，cin将不断读取，直到遇到空白；也就是说，cin将读取3、8、句点以及5，使得空格成为输入队列中的下一个字符，然后将这个4个字符的字符编码存储到数组中，并在末尾加上空字符；这里不需要转换；
  • 面向行输入的char数组：与上述类似，不同的是，cin将读取到换行符之前；这里同样不需要转换；

输出与此同理；

写入文本文件

需要指出，用于文件输出需要提供头文件fstream——其中定义了用于输出的ofstream类；

需要声明一个或多个ofstream变量或称对象；

必须指明命名空间std；

使用opne()方法将ofstream与文本文件关联起来；

使用完文件之后，应使用close()方法将其关闭；

示例如下：

#include<iostream>
#include<fstream>
using namespace std;

int main()
{
    char automobile[50];
    int year;
    double a_price;
    double b_price;

    ofstream outfile;
    outfile.open("carinfo.txt");

    cout << "Enter the make and model of automobile: ";
    cin.getline(automobile, 50);
    cout << "Enter the model year: ";
    cin >> year;
    cout << "Enter the original asking price: ";
    cin >> a_price;
    b_price = 0.93 * a_price;

    cout << fixed;
    cout.precision(2);
    cout.setf(ios_base::showpoint);
    cout << "Make and model: " << automobile << endl;
    cout << "Year: " << year << endl;
    cout << "Was asking $" << a_price << endl;
    cout << "Now asking $" << b_price << endl;

    outfile << fixed;
    outfile.precision(2);
    outfile.setf(ios_base::showpoint);
    outfile << "Make and model: " << automobile << endl;
    outfile << "Year: " << year << endl;
    outfile << "Was asking $" << a_price << endl;
    outfile << "Now asking $" << b_price << endl;

    outfile.close();
    return 0;
}

程序输出为两份，一份为命令行输出另一份为文本文件输出：

Make and model: Porches Machder
Year: 1973

Was asking $320000.00

Now asking $297600.00

但其输出内容一致；

需要指出：在创建好ofstream对象之后便可以像使用cout那样使用它，cout的任何方法都可以被ofstream对象使用；

另外，方法open()接受一个C-风格字符串作为参数，可以是字面字符串，也可以是存储在数组中的字符串；

语句outfile.open("carinfo.txt");的功能为将文本文件与ofstream对象相关联，如果文件目录中没有此文件，将创建一个该名称的文本文件，如果存在该文件，则open()将首先截断该文件，即将其长度截断到0——丢弃原有的内容，然后将新的输出加入到该文件中；

读取文本文件

与写入文本文件对应，读取文本文件同样需要提供头文件：fstream；

需要创建一个或多个ifstream对象；

须指明名称空间std；

使用open()与close()关联文本文件和关闭文本文件；

示例如下：

#include <iostream>
#include <fstream>
#include <cstdlib>
using namespace std;

int main()
{
    //获取文本文件名
    char filename[100];
    cout << "Enter the filename: ";
    cin.getline(filename,100);

    //关联文本文件
    ifstream infile;
    infile.open(filename);

    //判断是否读取了文本
    if (!infile.is_open())
    {
        cout << "Could not open this file " << filename << endl;
        cout << "Program terminating.\n";
        exit(EXIT_FAILURE);
    }

    //统计文本中int类型数值和double类型数值的数量
    double value;
    double sum = 0.0;
    int count = 0;

    infile >> value;
    while (infile.good())//如果成功读取则返回1，失败读取则返回0
    {
        ++count;
        sum += value;
        infile >> value;
    }
    //判断是否成功读取了所有数据
    if (infile.eof())
    {
        cout << "End of file reached.\n";//如果读取到文件末尾则返回true
    } else if (infile.fail())
    {
        cout << "Input terminated by date mismatch.\n";//如果遇到类型不匹配则返回true
    } else
    {
        cout << "Input terminated for unknown reason.\n";
    }
    //进行最后输出
    if (count == 0)
    {
        cout << "No date processed.\n";
    } else
    {
        cout << "Items read: " << count << endl;
        cout << "Sum: " << sum << endl;
        cout << "Average: " << sum / count << endl;
    }
    infile.close();
    return 0;

}

程序输出为：

Enter the filename:D:\cpp_clion\Chap06\infile\scores.txt
End of file reached.
Items read: 12
Sum: 204.5
Average: 17.0417

通过上例，有如下要点：

  1. is_open()方法用来判断是否成功打开文本文件；如果文件被成功打开，is_open()返回true；

  2. 函数exit()用来终止程序，其被定义在头文件cstdlib中；

  3. 声明一个ifstream对象之后便可以像使用cin那样使用它；

  4. 方法good()指出最后一次读取输入的操作是否成功；

  5. good() 等价于 !(bad() || eof() || failed())；

  6. `infile >> value;

    while (infile.good())
{
    .....
    infile >> value;
}`

    这部分代码可以精简为：

    `while (infile >> value)
{
    ......
}`

    表达式infile >> value的结果为infile，在需要一个bool类型的值的情况下，infile的结果为infile.good()；如果要计算该表达式的值，程序必须首先试图将一个值读取到value中；

Go语言-map

本文最后更新于：2021年2月9日 凌晨

map

Go语言中，map是一种无序键值对(key-value)的数据结构；为引用类型，须初始化后使用；

引用类型默认值为nil

map的声明与初始化

共有两种方法：

//map的声明与初始化
//先声明后初始化
var a map[string]int        //声明一个map，键为string类型，值为int类型
a = make(map[string]int, 5) //初始化
fmt.Printf("%T  %v", a, a)  //map[string]int  map[]
a["bili"] = 4
a["ten"] = 3
a["you"] = 2
a["aq"] = 1
fmt.Printf("a：%#v", a)
//a：map[string]int{"aq":1, "bili":4, "ten":3, "you":2}

//声明的同时初始化
x := map[int]string{
	1: "C++",
	2: "Java",
	3: "Python",
	4: "Golang",
}
fmt.Printf("x：%#v", x)
//x：map[int]string{1:"C++", 2:"Java", 3:"Python", 4:"Golang"}

map的使用

判断某个键是否存在于map中

直接代码展示：

//判断某个键是否存在
p := map[string]int{
	"Jack":  1,
	"Tom":   2,
	"Susan": 3,
}
value, ok := p["Jack"]
fmt.Println(value, ok) //1 true
value1, ok1 := p["Java"]
fmt.Println(value1, ok1) //0 false

如果键(key) 存在于map中，则value返回此键(key)对应的值(value)，ok返回true；否则value返回0值（不一定是数字0），ok返回false；

map的遍历

p := map[string]int{
	"Jack":  1,
	"Tom":   2,
	"Susan": 3,
}

for k, v := range p {
	fmt.Println(k, v)
}
//Tom 2
//Susan 3
//Jack 1

要注意的是，因为map本身无序的，所以打印出来的键值对顺序与添加的顺序无关

当然，也可以只遍历key（不写value即可）,也可以只遍历value（将k替换为_）；

按照顺序遍历map

由于map为无序结构，所以按照常规方法遍历得到的也是无序的；如果想map按照固定顺序遍历，需要采取一定手段：如借助于切片的sort包：

//按照固定顺序去遍历map
var students = make(map[int]string, 100) //创建一个容量为100的map

//添加50个数据
for i := 0; i < 50; i++ {
	key := rand.Intn(50)               //生成0~50的随机整数
	value := fmt.Sprintf("stu%02d", i) //将i以字符串格式输出
	students[key] = value              //将生成的key与value装进map中
}

// for k, v := range students {
// 	fmt.Println(k, v)
// }

//1.将key值取出来存入到切片中
var s = make([]int, 0, 100) //创建一个容量为100，大小为0，[]int类型的切片
for key := range students { //从students中取出key并追加到s中
	s = append(s, key)
}
// fmt.Println(s)
//2.对切片中的值进行排序
sort.Ints(s)
// fmt.Println(s)
//3.按照排序之后的key对students进行排序
for _, key := range s {
	fmt.Println(key, students[key])
}

map中的删除操作

使用内置的delete()函数

t := map[string]bool{
	"S": true,
	"T": false,
	"P": true,
	"Q": false,
}
//删除"S" 键值对
delete(t, "S")
fmt.Println(t)
//map[P:true Q:false T:false]

map的切片与切片的map

切片为数据集合，map为一种数据结构，所以，一种元素类型为map的切片也是存在的；

不同于其他元素类型的切片，元素类型为map的切片需要有两步初始化，即切片(Slice)初始化和map初始化

//元素类型为map的切片
var mapSlice = make([]map[string]int, 8, 8) //完成切片的初始化，此时默认为nil
mapSlice[0] = make(map[string]int, 8)       //完成map的初始化

mapSlice[0]["Java"] = 100 //对map添加键值对
fmt.Println(mapSlice) 	  //[map[Java:100] map[] map[] map[] map[] map[] map[] map[]]

与此对应，也存在值为map的切片

//值为切片的map
var sliceMap = make(map[string][]int, 8)
fmt.Println(sliceMap)
sliceMap["中国"] = make([]int, 8) //切片的初始化
sliceMap["中国"][0] = 100
sliceMap["中国"][1] = 200

//遍历sliceMap
for k, v := range sliceMap {
	fmt.Println(k, v)
}

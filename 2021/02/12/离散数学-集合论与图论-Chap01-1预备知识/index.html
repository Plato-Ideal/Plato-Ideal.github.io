Chap01 集合

课程概览

集合论

研究对象

集合、关系、函数、自然数、基数；

研究思想

以逻辑为基础、以集合为工具、表示和构造各种数学对象

研究内容
  • 集合的基本概念：集合之间的关系、运算、恒等式
  • 二元关系：表示、性质、函数、等价关系、序关系
  • 自然数：皮亚诺系统、自然数的运算、性质
  • 基数：有穷集与无穷集、基数的比较
  • 序数：良序、超限归纳法

图论

研究对象

由顶点和边构成的图

研究思想

以集合论为基础、以图为工具、为各种二元关系建立模型

研究内容

  • 图的基本概念：连通性、矩阵表示、带权图
  • 欧拉图、哈密顿图：边和顶点的遍历
  • 树：表示层级关系组织
  • 平面图：判定、表示、性质
  • 图的着色：各种调度问题的模型
  • 独立集、支配集、覆盖集、匹配：各种应用问题

命题逻辑预备知识

命题公式

联结词

  • p、q、r、...表示原子命题(或简单命题)
    1表示命题的真值为真，0表示命题的真值为假

  • $\urcorner$ 称为否定联结词；$\urcorner$p称为p的否定式；

    当p为0时，$\urcorner$p为1；当p为1时，$\urcorner$p为0；

  • $\wedge$称为合取联结词；p$\wedge$q称为p与q的合取式；

    即p与q同为1时，p$\wedge$q为1，否则为0；

  • $\vee$称为析取联结词；p$\vee$q称为p与q的析取式；

    只有p、q同为0时，p$\vee$q才为0，否则为1；

  • $\rightarrow$称为蕴含联结词；p$\rightarrow$q称为p与q 的蕴含式

    其意为：若p则q；

    | p | q | p$\rightarrow$q |
    | :—: | :—: | :——————-: |
    | 0 | 0 | 1 |
    | 0 | 1 | 1 |
    | 1 | 0 | 0 |
    | 1 | 1 | 1 |

    表示一个推理过程，由p得出q；与实际情况不一定能吻合；

    可以理解为p<=q，如果p>q则为假，其余为真；

  • $\leftrightarrow$称为等价联结词；p$\leftrightarrow$q称为p与q的等价式；

    只有当p、q同为0或同为1时，p$\leftrightarrow$q才为真，否则为假；

命题公式
  1. 单个命题变元（或常元）是命题公式；（pqr和01）
  2. 若A是命题公式，则 $\urcorner$A 也是；
  3. 若A，B是命题公式，则（A$\wedge$B），（A$\vee$B），（A$\rightarrow$B），（A$\leftrightarrow$B）也是；
  4. 只有有限次应用形成的符号串才是；（命题公式是由长度的）

没有联结词的称为简单命题，有联结词的称为复合命题；

联结词的优先级

​ $\urcorner$ > $\wedge$ = $\vee$ > $\rightarrow$ = $\leftrightarrow$

真值表：

1 0 0 0 1

赋值 赋值 可满足式 矛盾式（永假式） 重言式（永真式）
  • 赋值：对变元取值；
  • 可满足式：至少有一行赋值使得结果为真；
  • 矛盾式（永假式）：所有赋值的结果均为假；
  • 重言式（永真式）：所有赋值的结果均为真；

其中，这三种分类不是相互排斥的；可满足式与矛盾式相互排斥，任意一种公式不是可满足式就是矛盾式；重言式是一种特殊的可满足式；

等值演算

等值式：A$\leftrightarrow$B

(p$\rightarrow$q) $\leftrightarrow$ ($\urcorner$p$\vee$q)

基本 等值式

  1. 幂等律：A$\leftrightarrow$A$\vee$A，A$\leftrightarrow$A$\wedge$A

  2. 交换律：A$\vee$B$\leftrightarrow$B$\vee$A，A$\wedge$B$\leftrightarrow$B$\wedge$A

  3. 结合律：（A$\vee$B）$\vee$C $\leftrightarrow$ A$\vee$（B$\vee$C），（A$\wedge$B）$\wedge$C $\leftrightarrow$ A$\wedge$（B$\wedge$C）

  4. 分配律：A$\vee$（B$\wedge$C） $\leftrightarrow$ （A$\vee$B）$\wedge$ （A$\vee$C），A$\wedge$（B$\vee$C）$\leftrightarrow$ （A$\wedge$B）$\vee$ （A$\wedge$C）

  5. 德·摩根律：$\urcorner$（A$\vee$B）$\leftrightarrow$ $\urcorner$A$\wedge\urcorner$B

    验证：真值表

    | p | q | $\urcorner$（A$\vee$B） | $\urcorner$A$\wedge$$\urcorner$B |
    | :—: | :—: | :——————————-: | :———————————————: |
    | 0 | 0 | 1 | 1 |
    | 0 | 1 | 0 | 0 |
    | 1 | 0 | 0 | 0 |
    | 1 | 1 | 0 | 0 |

  6. 吸收律：A$\vee$（A$\wedge$B）$\leftrightarrow$ A，A$\wedge$（A$\vee$B）$\leftrightarrow$ A

  7. 零律：A$\vee$1 $\leftrightarrow$ 1，A$\wedge$0$\leftrightarrow$0 （互为对偶）

    1是或的零元；0是与的零元；

  8. 同一律：A$\vee$0$\leftrightarrow$A，A$\wedge$1$\leftrightarrow$A （互为对偶）

    1是与的单位元；0是或的单位元；

  9. 排中律：A$\vee$$\urcorner$A$\leftrightarrow$1 （有争议）（与矛盾律构成互为对偶）

  10. 矛盾律：A$\wedge$$\urcorner$A$\leftrightarrow$0

    PS：对偶原理：$\vee$-$\wedge$互换，0和1互换

  11. 双重否定律：$\urcorner$$\urcorner$A$\leftrightarrow$A

  12. 蕴涵等值式：A$\rightarrow$B$\leftrightarrow$$\urcorner$A$\vee$B

  13. 等价等值式：A$\leftrightarrow$B $\leftrightarrow$ （A$\rightarrow$B）$\wedge$（B$\rightarrow$A）

  14. 等价否定等值式：（A$\leftrightarrow$B） $\leftrightarrow$ （$\urcorner$A$\leftrightarrow$$\urcorner$B）

  15. 假言易位：（A$\rightarrow$B） $\leftrightarrow$ （$\urcorner$B$\rightarrow$ $\urcorner$A）

  16. 归谬论：（A$\rightarrow$B）$\wedge$（A$\rightarrow$$\urcorner$B）$\leftrightarrow$ $\urcorner$A

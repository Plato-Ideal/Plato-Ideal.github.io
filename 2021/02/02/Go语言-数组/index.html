Go语言-数组

本文最后更新于：2021年2月9日 凌晨

数组（Array）

数组是同一种数据类型元素的集合。在Go语言中，数组一经声明即确定，仅可以修改数组成员，但是其大小不能变。

var a [4]int //定义一个长度为4存放int类型的数组
var b [3]int //定义一个长度为3存放int类型的数组

注意：此时，a与b是不同的类型，即a与b存放的数据类型不一样，因为数组的长度也是类型的一部分。

数组类型由两部分组成：一为数组长度，二为其内存放元素的类型

数组的初始化

代码展示：

//初始化
a = [4]int{0, 1, 2, 3}
b = [3]int{3, 6, 9}

特别地，如果定义数组的长度比实际初始化的大，那么，其空缺部分会由默认值补齐

//数组补全
var a [4]int
var b [4]float64
var c [4]bool
var d [4]string

a = [4]int{1, 2}
b = [4]float64{1.1, 2.2}
c = [4]bool{true, true}
d = [4]string{"hello", "world"}

fmt.Println(a) //[1 2 0 0]
fmt.Println(b) //[1.1 2.2 0 0]
fmt.Println(c) //[true true false false]
fmt.Println(d) //[hello world  ]

数组初始化的另一种方法：

//声明与初始化的另一种方法
var a = [4]int{1, 2, 3, 4} //声明附带初始化
fmt.Println(a)

数组初始化的第三种方法：

//声明与初始化的第三种写法
var b = [...]int{1, 2, 3, 4, 4, 5, 3, 3, 5, 73, 75}
//...表示让编译器去计算数组的长度
fmt.Println(b)

数组初始化的第四种方法：

//声明与初始化的第四种写法:根据索引值初始化
var c [20]int
c = [20]int{10: 1, 12: 4} //表示索引值为10的位置上的数为1
fmt.Println(c)

数组的基本使用

打印指定索引位置上的值

var d [10]int
d = [10]int{1: 2, 2: 4, 4: 8, 8: 16}
fmt.Printf("数组d索引4上的数为:%d", d[4])

遍历数组（一）

//遍历数组
var d [10]int
d = [10]int{1: 2, 2: 4, 4: 8, 8: 16}

for i := 0; i < len(d); i++ {
	fmt.Println(d[i])
}

遍历数组（二）

//遍历数组(二)用for range循环
var d [10]int
d = [10]int{1: 2, 2: 4, 4: 8, 8: 16}

for index, value := range d { //index即索引，value即该索引对应的值
	fmt.Println(index, value) //value的位置也可以写 d[index]
}

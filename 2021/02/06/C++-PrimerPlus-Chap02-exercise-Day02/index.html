Chap2 复习

  1. C++模块叫什么？

    通常来说，C++中的模块一般是指函数；

  2. 下面的预处理器编译指令是做什么用的？

    #include<iostream>

    在源代码被编译之前，预处理器将iostream文件添加到源文件中，形成一份复合文件，一起交由编译器进行编译；iostream中封装着有关输入/输出的定义；

  3. 下面的语句是做什么用的？

    using namespace std;

    这是一条using编译指令；表示启用名称空间std；包括cout、cin等一系列预定义对象都被定义在其中；

  4. 什么语句可以用来打印”Hello World!”然后开始新的一行？

    cout << “Hello World!” << endl;

  5. 什么语句可以用来创建名为cheeses的整数变量？

    int cheeses;

  6. 什么语句可以将32赋值给变量cheeses？

    cheeses = 32;

  7. 什么语句可以用来将键盘输入的值读入变量cheeses中？

    cin >> cheeses;

  8. 什么语句可以打印”We have X varieties of cheese,”，其中X为cheeses的当前值？

    cout << “We have “ << cheeses << “ varities of cheese,”

  9. 下面的函数原型指出了关于函数的那些信息？

    int froop(double t);

    返回值为一个整数，函数名为froop，接收一个double类型的变量；

    void rattle(int n);

    无返回值，接收一个整数；

    int prune(void);

    返回一个整数，不接收参数；

  10. 定义函数时，在什么情况下不必使用关键字return？

    没有返回值的时候，不必使用return；如void simon()；

  1. 编写一个程序，让用户输入其年龄，然后显示该年龄包含多少个月；

    #include<iostream>
using namespace std;
int yeartomonth(int);
//编写一个程序，让用户输入其年龄，然后显示该年龄包含多少个月；

int main()
{
    cout << "Enter your age: ";
    int userage; //声明一个用户年龄用来接收用户输入
    cin >> userage;
    cout << endl;
    cout << "Your age include " <<  yeartomonth(userage) <<" months." << endl;
    return 0;
}

int yeartomonth(int year)
{
    return year * 12;//创建一个转换函数
}

  1. 编写一个程序，让其中的main()函数调用一个用户自定义函数，并且完成温度转换；（华氏温度 = 1.8 * 摄氏温度 + 32.0）

    #include<iostream>
using namespace std;
double convert_temp(double);

int main()
{
    int t;//接收温度
    cout << "Enter the tempreture:(C)";
    cin >> t;
    cout << endl << "The coverted temperture is "; 
    cout << convert_temp(t) << "F." << endl;
    return 0;
}

double convert_temp(double she)
{
    return (she * 1.8 + 32.0);//输入一个摄氏温度，返回一个华氏温度
}

  1. 编写一个程序，其main()函数调用一个用户自定义函数，完成光年和天文单位的转换（1 光年 = 63240天文单位）；

    #include<iostream>
using namespace std;
int convert_lightyear(int); //函数原型 function prototype

int main()
{
    int l;//接收光年
    cout << "输入光年长度: ";
    cin >> l;
    cout << endl << "转换后的长度为："; 
    cout << convert_lightyear(l) << " 天文单位" << endl;
    return 0;
}

int convert_lightyear(int light)
{
    return (light * 63240);//输入一个光年单位，返回一个天文单位
}

  1. 编写一个程序，要求用户输入小时数和分钟数。在main()函数中，将这两个函数传递给一个void函数，后者以这样的形式显示着两个值：

    Enter the numbers of hours: 9

    Enter the numbers of minutes: 28

    Time: 9:28

    #include<iostream>
using namespace std;
void showtime(int,int); //function prototype

int main()
{
    int hour,minite;
    cout << "Enter the numbers of hours:";
    cin >> hour;
    cout << endl << "Enter the numbers of minutes:";
    cin >> minite;
    cout << endl;
    showtime(hour,minite); //直接调用即可
    cout << endl;
}

void showtime(int hours,int minites)
{
    cout << "Time:" << hours << ":" << minites << endl;
}

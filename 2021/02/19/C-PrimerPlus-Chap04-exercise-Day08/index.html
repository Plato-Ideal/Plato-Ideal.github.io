  1. 编写一个C++程序，如下述输出示例所示的那样请求并显示信息；

    What’s your first name? Betty Sue

    What is you last name? Yewe

    What letter grade do you deserve? B

    What is your age? 22

    Name: Yewe,Betty

    Grade:C

    Age:22

    需注意，程序将下调成绩，默认只有A、B、C；

    #include <iostream>
using namespace std;

int main()
{
    char first_name[20];
    char last_name[20];
    int age;
    char grade;

    cout << "What's your first name?";
    cin.getline(first_name,19);
    cout << "What's your last name?";
    cin.getline(last_name,19);
    cout << "What letter grade do you deserve?";
    cin >> grade;
    cout << "What's your age?";
    cin >> age;
    cout << endl;

    cout << "Name: " << last_name << "," << first_name << endl;
    cout << "Grade: " << char(grade + 1) << endl;
    cout << "Age: " << age << endl;
}

    程序输出为：

    What’s your first name?Betty Yewe
    What’s your last name?Yewe
    What letter grade do you deserve?B
    What’s your age?22

    Name: Yewe,Betty Yewe
    Grade: C
    Age: 22

  1. 结构CandyBar包含了3个成员。第一个成员存储了糖块的品牌；第二个成员存储糖块的重量（可以有小数）；第三个成员存储了糖块的卡路里含量（整数）。请编写一个程序，声明这个结构，创建一个名为snack的CandyBar变量，并将其成员分别初始化为“Mocha Munch”、2.3和350。初始化应在声明snack时进行。最后，程序显示snack变量的内容。

    #include <iostream>
using namespace std;

struct candybar
{
    string brand;//品牌
    float weight;//重量
    int calo;//卡路里
};

int main()
{
    candybar snack = {"Mocha Munch", 2.3, 350};
    cout << "The brand: " << snack.brand;
    cout << "The weight: " << snack.weight;
    cout << "The energy: " << snack.calo;

    return 0;
}

    程序输出为：

    The brand: Mocha Munch

    The weight: 2.3

    The energy: 350

  1. William Wingate 从事比萨饼分析服务。对于每个披萨饼，他都需要记录下列信息：

    • 披萨饼公司的名称，可以有多个单词组成。
    • 披萨饼的直径。
    • 披萨饼的重量。

    请设计一个能够存储这些信息的结构，并编写一个使用这种结构变量的程序。程序将请求用户输入上述信息，然后显示这些信息。请使用cin（或它的方法）和cout。

    #include <iostream>
#include <string>
using namespace std;

struct pizza_analyze
{
    string company;//公司名称
    int diameter_inch;//披萨直径
    double weight;//披萨重量
};

int main()
{

    pizza_analyze pizza;
    cout << "Enter the name of company:";
    getline(cin,pizza.company);
    cout << "Enter the dia of pizza:";
    cin >> pizza.diameter_inch;
    cout << "Enter the weight of pizza:";
    cin >> pizza.weight;

    pizza_analyze * pa = &pizza;//单纯为了测试自己对于结构指针的掌握，简单的办法是通过成员运算符直接调用

    cout << "Company: " <<  pa->company << endl;
    cout << "Diameter: " << pa->diameter_inch << endl;
    cout << "Weight: " << pa->weight << endl;

    return 0;
}

    程序输出为：

    Enter the name of company:Beauriful Pizza
    Enter the dia of pizza:9
    Enter the weight of pizza:500

Company: Beauriful Pizza
Diameter: 9
Weight: 500

  1. 完成编程练习7，但使用new来为结构分配内存，而不是声明一个结构变量。另外，让程序在请求输入比萨饼公司名称之前输入比萨饼的直径。

    #include <iostream>
#include <string>
using namespace std;

struct pizza_analyze
{
    string company;//公司名称
    int diameter_inch;//披萨直径
    double weight;//披萨重量
};

int main()
{

    pizza_analyze * pizza = new pizza_analyze;//使用new创建结构指针

    cout << "Enter the weight of pizza:";//用户输入
    cin >> pizza ->weight;
    cin.get();
    cout << "Enter the name of company:";
    getline(cin,pizza -> company);
    cout << "Enter the dia of pizza:";
    cin >> pizza->diameter_inch;
    cout << endl;

    cout << "Company: " <<  pizza->company << endl;
    cout << "Diameter: " << pizza->diameter_inch << endl;
    cout << "Weight: " << pizza->weight << endl;

    delete pizza;//释放内存

    return 0;
}

    程序输出为：

    Enter the weight of pizza:450
    Enter the name of company:Atlatic
    Enter the dia of pizza:11

    Company: Atlatic
    Diameter: 11
    Weight: 450

  1. 编写一个程序，让用户输入三次40码跑的成绩（如果您愿意，也可让用户输入40米跑的成绩），并显示次数和平均成绩。请使用一个array对象来存储数据（如果编译器不支持array类，请使用数组）。

    #include <iostream>
#include <array>
using namespace std;

int main()
{
    array<float,3>score;

    cout << "Enter your first score:";
    cin >> score[0];
    cout << "Enter your second score:";
    cin >> score[1];
    cout << "Enter your third score:";
    cin >> score[2];
    cout << endl;

    float average = (score[0] + score[1] + score[2]) / 3;

    cout << "It's your score:\n";
    cout << "First Time: " << score[0] << endl;
    cout << "Second Time: " << score[1] << endl;
    cout << "Third Time: " << score[2] << endl;
    cout << "The average score: " << average << endl;

    return 0;

}

    程序输出为：

    Enter your first score:5.37
    Enter your second score:5.33
    Enter your third score:4.96

***It's your score:
First Time: 5.37
Second Time: 5.33
Third Time: 4.96
The average score: 5.22***

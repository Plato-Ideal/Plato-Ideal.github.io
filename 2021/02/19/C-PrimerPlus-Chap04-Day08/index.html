4.10 数组替代品

模板类vector、array是数组的替代品

vector

vector模板类是一种动态数组；在C++98中定义；可在运行阶段设置vector对象的长度，可以在末尾添加数据，也可以在中间插入数据；实际上，vector也是用new和delete来管理内存；

使用vector需要注意，vector包含在头文件vector中，而头文件vector包含在名称空间std中；

vector<int>vi;

int n;
cin >> n;
vector<float>vf(n);

如上是两种使用vector的方法，第一个vector将创建一个长度为0的int数组，第二个vector使用变量n来声明float数组的长度；

array

不同于vector的是，array模板类将数组存储在栈中；它在C++11中定义；array对象的长度是固定的；要创建array对象，需要包含头文件array；

array<int,5>ai;
array<double,4>ad = {1.0, 2.0, 3.0, 4.0};

特别要注意的一点是，array不支持使用变量声明对象长度；

在C++11中，可将列表初始化用于vector和array对象；

数组、vector和array

这里比较一下三者的区别：

  • 无论是数组、vector对象还是array对象，都可以使用标准数组表示法来访问各个元素；
  • 数组和array对象存储在栈中，而vector对象存储在自由存储区或堆中；
  • array对象可以赋值给另一个array对象，而数组必须逐元素复制；
//数组
double d1[4] = {1.0, 2.0, 3.0, 4.0};

//vector
vector<int>i1(4);
i1[0] = 1;
i1[1] = 2;
i1[2] = 3;
i1[3] = 4;


array<float,4>f1 = {1.4,2.8,3.7,5.9};
array<float,4>f2;
f2 = f1;

cout << "d1[0]:" << d1[0] << " at:" << &d1[0] << endl;
//d1[0]:1 at:0x61fe98
cout << "i1[0]:" << i1[0] << " at:" << &i1[0] << endl;
//i1[0]:1 at:0xf77010
cout << "f1[0]:" << f1[0] << " at:" << &f1[0] << endl;
//f1[0]:1.4 at:0x61fe7c
cout << "f2[0]:" << f2[0] << " at:" << &f2[0] << endl;
//f2[0]:1.4 at:0x61fe6c

注意：数组信息可以通过修改索引的方式改变信息存储的位置，即使是数组外，但这种方式将导致不确定的结果：

d1[-1] = 3.14;
   cout << "d1[-1]:" << d1[-1] << " at:" << &d1[-1] << endl;
   //d1[-1]:4.20105e-304 at:0x61fe70

所以，可以通过使用vector和array对象的成员函数at()来规避越界错误；

在使用成员函数at()时，程序会捕捉非法索引，继而中断程序；但代价是程序的运行时间会延长；

2.1 进入C++

代码分析

#include<iostream>
int main()
{
    using namespace std;
    cout << "Hello World~";
    cout << endl;
    cout << "Once again~" << endl;
    return 0;
}

这个例子主要包括以下元素：

  • 注释 \\ ；
  • 预处理器编译指令 #include；
  • 函数头 int main()；
  • 编译指令 using namespace；
  • 函数体 用{}括起来的部分；
  • 使用C++的cout工具显示消息的语句；
  • 结束main()函数的return语句；

main()

函数定义（function definition）

包含函数头(function heading)和函数体(由{}括起来的部分)；

其中函数头对函数与程序其他部分之间的接口做了总结；函数体指出函数应该做什么；

函数头

在这个例子中，函数头为 int main();

其中，int部分称作函数返回类型，描述了函数返回给调用它的函数的信息，int表明此函数返回一个int类型的值

main()中()部分为形参列表(argument list)，或称参数列表(parameter list)，它描述了从调用函数传递给被调用函数的信息；当括号为空时，表明函数不接受任何参数

特别注意的是：main()函数在C++程序中是必要的；其提供了程序的入口，程序通常从这里开始；

例外：

  • Windows编程中，由于dll模块(动态链接库)不是独立程序，所以不需要main()函数；
  • 专用环境的程序——如机器人中的控制芯片；
  • 提供框架程序的编程环境——会调用一些非标准函数，有隐藏main(),调用非标准函数；

函数体

包含语句与结束函数；

语句必须以终止符(terminator)即;结尾；

结束函数(return statement)也叫返回语句，负责结束函数；ANSI/ISO C++标准规定：如果一个main()函数中，编译器直到函数末尾都没有遇见rerun语句，则默认为以return 0;结尾

C++注释

一共有两种，一种是C++标准注释，即双斜杠(//)打头的写法；另一种则是C风格注释，即/**/；可以跨多行进行注释；

C++预处理器和iostream文件

#include即预编译指令，它在主编译之前对源文件进行操作；在这里，预处理器将iostream文件与源文件组合成一个复合文件；

iostream中包含了C++的输入输出方案，为了使用这些定义，需要将iostream文件中的内容添加到源程序中；

特别注意：如果使用cin和cout进行输入输出的程序必须包含文件iostream

头文件名

如iostream这样的文件被称作包含文件(include file)，也称作头文件(header file)

头文件类型有如下四种：

  1. C++旧式风格：以.h结尾；C++程序可以使用
  2. C旧式风格：以.h结尾；C、C++程序均可使用
  3. C++新式风格：没有扩展名；C++程序可以使用，使用namespace std
  4. 转换后的C：加上前缀C，没有扩展名；C++程序可以使用，可以使用不是C的特性

名称空间

名称空间作为一种附加信息用来区分不同库中相同名称的函数、类和变量等；

其使用分为两种：

其一：在文件开头使用using编译指令；

其二：在所需要使用的函数前面加上名称空间，如std::endl

#include<iostream>
int main()
{
    // using namespace std;
    std::cout << "Hello World~";
    std::cout << std::endl;
    std::cout << "Once again~" << std::endl;
    return 0;
}

cout

cout是一个预定义对象，主要作用为输出字符串、数字、单个字符等；

它同插入运算符<<一起工作

从C++角度描述cout输出，可以这样解释：将输出看作流，cout表示这种流，<<将数据插入到流中；所以与其说程序显示了消息，莫不如说它将一个字符串插入到了输出流中；

cout特点：

  • 可扩展的(extensible)：可以重新定义<<运算符，使得cout能够识别和显示所开发的数据类型；

特殊用法：

cout可以将数据进行合并输出：

cout << "Now you have " << carrots << "carrots." << endl;

或者：

cout << "Now you have " 
cout << carrots 
cout << "carrots." 
cout << endl;

或者：

cout << "Now you have " 
	<< carrots 
	<< "carrots." 
	<< endl;

endl VS \n

endl是一个特殊符号，称为控制符(manipulator)，表示重启一行；endl在头文件iostream中定义，位于命名空间std中

另一种换行方案为"\n"；被称为换行符；它与endl功能相同，大多数时候，我们可以把二者视为一致；在细微处，二者有不同：

cout << endl;		//与下相等
cout << "\n" <<flush;

具体使用视情况而定

C++代码格式

概念

  • 标记(token)：一行代码中不可分割的元素；
  • 空白(white space)：空格、制表符、回车；

规则

  1. 每条语句占一行；
  2. 每个函数都有一个开始花括号和一个结束花括号，这两个花括号各占一行；
  3. 函数中语句都相对于花括号进行缩进；
  4. 与函数名称相关的圆括号周围没有空白；

2.2 C++语句

作为例子的代码：

#include<iostream>

int main()
{
    using namespace std;

    int carrots; //declare a integer variable

    carrots = 25; //assign a value to the variable
    cout << "I have";
    cout << carrots;
    cout << "carrots.";
    cout << endl;
    carrots = carrots - 1;
    cout << "Crunch,crunch.Now I have" << carrots << "carrots." << endl;
    return 0;

}

声明语句和变量

语句int carrots;是一条声明语句，它指出了需要的内存空间及相应的内存单元名称；其指令要求计算机做两件事：

  1. 指出程序需要足够的存储空间来存储一个整数；
  2. 给存储单元指定名称；

程序中的声明语句叫做定义声明(defining declaration)语句，简称为定义(definition)；声明不一定是定义；

为什么要声明变量？

短期来看，貌似不声明对程序员更为友好，但长期来看，不声明变量具有较大的潜在风险，比如在不知情的情况下创建一个新变量；

赋值语句

carrots = 25;为一个赋值语句，它表明将整数25赋给变量carrots所表示的内存单元；

其中=称作赋值运算符；

特性

可以连续使用=：

int fly;
   int butter;
   int slowed;
   fly = butter = slowed = 23;

这样写在C++中是合法的；

规则

赋值运算符从右往左进行；

2.3 其他C++语句

例程：

#include<iostream>

int main()
{
    using namespace std;

    int carrots;

    cout << "How many carrots do you have?" << endl;
    cin >> carrots;
    cout << "Here are two more. ";
    carrots = carrots + 2;
    cout << "Now you have " << carrots << "carrots." << endl;
    return 0;
}

cin

cin承担了将键盘输入赋值给变量的任务；

正如cout描述一样，我们同样可以把cin也看作一种流，不同的是，cin流向的是程序本身，与>>符一同工作；

类简介

类是用户定义的一种数据类型，定义类需要描述他能够表示什么信息和可对数据执行哪些操作。

类之于对象就行类型之于变量；

类描述了一种数据类型的全部属性（包括可以使用它执行的操作），对象是根据这些描述创造的实体；

Go语言-new与make

本文最后更新于：2021年2月9日 凌晨

new vs make

  • make是用来给引用类型的数据做初始化；比如：slice、map、channel
  • new是用来给值类型做初始化；比如:pointer；通常情况下，只有初始化值类型指针时才会用到new，其他时候一般不用new；
func main() {
	//new的用法
	var a = new(int) //得到一个int类型的指针
	fmt.Println(a)   //0xc0000a2058 此时a为内存地址
	fmt.Println(*a)  //0
	*a = 10          //根据内存地址找到值，然后重新赋值
	fmt.Println(*a)  //10

}

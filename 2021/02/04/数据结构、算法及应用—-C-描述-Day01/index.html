Chap01

1.2 函数与参数

1.2.1 传值参数

参数的传递机制：

函数在执行前，将实参（actual parameter）的值复制一份——由复制构造函数（copy constructor）完成——传递给形参（formal parameter），此时的形参就可以称之为传值参数（value parameter）。

当程序运行结束，由形参类型的析构函数（destructor）负责释放形式参数；

#include<iostream>
using namespace std;

int abc(int a,int b,int c) //形参
{
    return a + b * c;
}

int main()
{
    cout << abc(2, 3, 4); //实参
}

1.2.2 模板参数

所谓模板参数（Function template），就是建立一个通用函数，不指明数据类型（包括返回值类型、形参类型、局部变量类型），而用一个虚拟的类型代替（实际上是用一个标识符来占位），等函数真正调用时，再根据传入参数逆推出真正的类型，而在这个过程中，这个通用函数就称为模板函数；

本质上反应了C++的重要特性：即值（value）和类型（Type）都可以参数化；

template<class T>
T abc1(T a,T b,T c)
{
    return a + b * c;
}

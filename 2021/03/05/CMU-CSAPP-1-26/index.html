Section 1

示例1

示例：$x^{2} >= 0$在计算机世界中有着截然不同的结果；

  • 对于浮点数来说，它总是正确的；
  • 但是对于整数int类型来说，结果就不一定了
cout << 40000 * 40000 << endl;//1600000000
cout << 50000 * 50000 << endl;//-1794967296

此时可以看到50000的平方出现了负数；

并非程序出了错，而是计算机在计算50000的平方的时候恰好得到了一个负数的格式；

示例2

数值运算是否符合加法交换律和结合律？

对于有符号整数和无符号整数来说是符合的；但是对于浮点数来说结果就不一定了；

cout << (1e20 + -1e20) + 3.14 << endl;//3.14
cout << 1e20 + (-1e20 + 3.14) << endl;//0

很显然，后者的结果不符合交换律；

上述两个例子共同说明了一个事实：它们用有限的位组合表示在数域中无限的数，所以必然要在实现上有所妥协：要么溢出，要么舍去；

通常来说，整数体现为溢出，浮点数体现为舍去；

Go语言-字符串操作

本文最后更新于：2021年2月9日 凌晨

字符串

Go语言中的字符串以原生数据类型（内置）出现，就像使用其他原生数据类型一样（如int、bool、float32、float64等）。其使用UTF-8编码（支持中文）。字符串中的值为双引号（""）中的内容。Go语言中单引号（''）中的内容代表字符——即一个单独的字母、汉字或者符号等

s := "Java number.1"
c1 := 'J'
c2 := 'a' //其后同理

字符串转义符

即\

  1. \t 制表符
  2. \\ 反斜线
  3. \" 双引号
  4. \' 单引号
  5. \r 回车符（返回行首）
  6. \n 换行符（直接跳到下一行的同列位置）

多行字符串

s := `
	天青色
	等烟雨
	而我在等你
`

反引号间的内容会原样输出，所有转义字符均无效

字符串中的常用操作

  • len(str) —— 求长度
  • + 或者fmt.Sprintf —— 拼接字符串
  • strings.Split —— 分割
  • strings.contains —— 判断是否包含
  • strings.HasPrefix,strings.HasSuffx —— 前缀、后缀判断
  • strings.Index(),strings.LastIndex() —— 子串出现的位置
  • strings.Join(a[]string,sep string) —— join操作
//字符串操作
	s3 := "Gopher"
	s4 := "Nomber.one"

	fmt.Println(len(s3)) //求s3长度

	fmt.Println(s3 + s4)              //字符串拼接其一
	ss := fmt.Sprintf("%s%s", s3, s4) //拼接字符串其二
	fmt.Println(ss)

	ret := strings.Split(ss, ".") //字符串分割
	fmt.Println(ret)

	fmt.Println(strings.Contains(ss, "Go")) //字符串包含判断

	fmt.Println(strings.HasPrefix(ss, "ph"))  //前缀判断
	fmt.Println(strings.HasSuffix(ss, "one")) //后缀判断

	fmt.Println(strings.Index(ss, "e"))     //子串索引判断
	fmt.Println(strings.LastIndex(ss, "e")) //子串索引判断

	fmt.Println(strings.Join(ret, "+")) //join操作
}

byte与rune型

byte为uint8的别名，只能表示ASCII码； rune为int32的别名，可以表示UTF-8的编码

发明byte与rune的目的：提高程序可读性。

类型转换

Go语言中只有强制类型转换，没有隐式类型转换，使用强转符()

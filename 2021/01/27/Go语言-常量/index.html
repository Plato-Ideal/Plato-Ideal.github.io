Go语言-常量

本文最后更新于：2021年2月9日 凌晨

常量

相对于变量，常量是恒定不变的值，多用于定义程序运行期间不会改变的那些值。其声明与变量类似，关键字为const，常量在定义的时候必须赋值。

声明与赋值

package main

import "fmt"

const pi = 3.1415926

//常量定义了之后不能修改

const ( //批量声明
	statusOK = 200.34
	notFound = 404
)

const (
	n1 = 100
	n2
	n3
)

func main() {

	fmt.Println(statusOK)
	fmt.Println(notFound)
}

注意：

  1. 批量声明的时候，如果第一行声明并且赋值，其后的行都没有赋值，只有常量名，那么其后行的值与第一行相同

iota

iota是go语言的常量计数器，只能在常量的表达式中使用

iota在const关键字时，将被重置为0、const中每新增一行常量声明，将使得iota技术一次（行索引）。

const(
	a1 = iota //0
    a2        //1
    a3        //2
    a4        //3
)

iota主要实现类似枚举的效果

  • 几个特殊的例子

    使用_跳过某些值

const ( //只要const出现，iota便重置为0
	b1 = iota //0
	b2        //1
	_         //2
	b3        //3
)

​ iota声明中间插队

const (
	c1 = iota //0
	c2 = 100  //100
	c3 = iota //2
	c4        //3

)

​ 多个常量声明在一行

const (
	d1, d2 = iota + 1, iota + 2 //d1 = 1 d2 = 2    iota = 0
	d3, d4 = iota + 1, iota + 2 //d3 = 2 d4 = 3    iota = 1
)

​ 定义数量级

const (
	_  = iota
	KB = 1 << (10 * iota) //iota = 1
	MB = 1 << (10 * iota) //iota = 2
	TB = 1 << (10 * iota) //iota = 3
	PB = 1 << (10 * iota) //iota = 4
)

​ _ 为忽略值，iota 赋值给_则将0去掉

​ 1 << (10 * iota) 则代表将1左移10位，即2^10 = 1024，其后同理

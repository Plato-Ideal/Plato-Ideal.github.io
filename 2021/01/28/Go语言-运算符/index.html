Go语言-运算符

本文最后更新于：2021年2月9日 凌晨

运算符

算术运算符

即+ - * / % 五种

//1.算术运算符
	a := 10.0
	b := 20.0
	fmt.Println(a + b)  //30
	fmt.Println(a - b)  //-10
	fmt.Println(a / b)  //0.5
	fmt.Println(a * b)  //200
	fmt.Println(10 % 4) //2

在Go语言中，++ -- 不是运算符，而是语句。

关系运算符

即== != > >= < <= 共六种。比较容易理解，不在此详述

逻辑运算符

  • && 逻辑AND运算符，如果两边操作都为TRUE，则为TRUE，否则为FALSE
  • || 逻辑OR运算符，如果两边操作有一个为TRUE，则为TRUE，否则为FALSE
  • ! 逻辑NOT运算符，如果条件为TRUE ,则为FALSE,否则为TRUE 
//3.逻辑运算符
	fmt.Println(1 > 2 && 2 < 3) //false
	fmt.Println(1 > 2 || 2 < 3) //true
	fmt.Println(!(2 < 3))       //false

位运算符

即& | ^ << >> ，对整数在内存中的二进制位进行操作

  • & 各对应的二进制位相与——两位均为1才为1
  • | 各对应的二进制位相或——两位有一个为1则为1
  • ^ 各对应的二进制位相异或——两位不一样则为1
  • << 左移n位就是乘以2的n次方（高位丢弃，低位补0）
  • >> 右移n位就是除以2的n次方
a := 34                      //0010 0010
b := 43                      //0010 1011
fmt.Printf("a&b= %b\n", a&b) //100010 => 34
fmt.Printf("a|b= %b\n", a|b) //101011 => 43
fmt.Printf("a^b= %b\n", a^b) //1001   => 9
c := 4 //0100
d := 5
fmt.Printf("c << d = %b\n", c<<d) //即4 * (2 ^ 5) = 128
//相当于将0100左移5位，即10000000
fmt.Printf("c >> d = %b\n", c>>d) //即4 / (2 ^ 5) = 0
//相当于将0100右移5位，即0

赋值运算符

  • = x = 2 即将2赋值给x
  • += x += 2 => x = x + 2
  • -= x -= 2 => x = x - 2
  • *= x = 2 => x = x 2
  • %= x%2 => x = x % 2
  • <<= x <<= 2 => x = x << 2
  • >>= x >>= 2 => x = x >> 2
  • &= x &= 2 => x = x & 2
  • |= x |= 2 => x = x | 2
  • ^= x ^= 2 => x = x ^ 2

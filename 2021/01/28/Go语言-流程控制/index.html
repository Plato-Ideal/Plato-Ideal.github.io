流程控制

if-else 分支结构

代码展示

//1.标准写法
s := 100
if s >= 90 {
	println("You're wonderful")
} else if s >= 70 && s < 90 {
	println("You're great")
} else {
	println("You're good")
}

//2.特殊写法

if s1 := 90; s1 >= 90 {
	println("You're wonderful")
} else if s1 >= 70 && s1 < 90 {
	println("You're great")
} else {
	println("You're good")
}

注意：Go语言中，if-else分支结构的花括号的左半部分（ { ）必须和if-else在一行，否则程序会报错！！！

//程序报错
if s12 := 90; s12 >= 90 
{
	println("You're wonderful")
} else if s12 >= 70 && s12 < 90 
{
	println("You're great")
} else 
{
	println("You're good")
}

//报错信息：
"message": "expected '{', found newline",

特别地，Go语言中可以只写if语句，if else 以及 else 都是可以丢弃的。

for（循环结构）

Go语言中的所有循环类型均可以使用for关键字来完成

代码展示

//for循环
//1.标准写法
for i := 0; i < 10; i++ {
	fmt.Println(i)
}

//2.省略初始语句，但是必须保留初始语句之后的分号
i := 0
for ; i < 10; i++ {
	fmt.Println(i)
}

//3.初始语句与结束语句都省略
var t int = 5
for t > 1 {
	fmt.Println(t)
	t--
}

break与continue

break与continue 都是跳出循环，但二者有所不同。break是直接结束for循环；continue则是跳出本次循环：

//break
for i := 0; i < 9; i++ {
	fmt.Print(i)
	if i == 4 {
		break
	}
} //结果：01234
fmt.Println()

//continue
for i := 0; i < 9; i++ {
	if i == 4 {
		continue
	}
	fmt.Print(i)
} //结果：01235678

特殊用法：无限循环（死循环）

//无限循环写法
for {
	//循环体语句
}

无限循环可以通过break,continue,goto,return,panic语句强制退出循环

switch case语句

对if-else分支结构的扩展

//switch-case
month := 4
switch month {
case 1:
	fmt.Println("一月")
case 2:
	fmt.Println("二月")
case 3:
	fmt.Println("三月")
case 4:
	fmt.Println("四月")
case 5:
	fmt.Println("五月")
case 6:
	fmt.Println("六月")
case 7:
	fmt.Println("七月")
case 8:
	fmt.Println("八月")
case 9:
	fmt.Println("九月")
case 10:
	fmt.Println("十月")
case 11:
	fmt.Println("十一月")
case 12:
	fmt.Println("十二月")
    default:
       fmt.Println("无效")
}

switch-case还可以一次判断多个值：

//一次判断多个值
num := 4
switch num {
case 1, 3, 5:
	fmt.Println("奇数")
case 2, 4, 6:
	fmt.Println("偶数")
}

case中也可以使用表达式

//case中使用表达式
age := 2
switch {
case age >= 18:
	println("You're adult!")
default:
	println("You're so young!")
}

goto（跳转到指定标签）

通常用来帮助简化代码使用，实现代码间的无条件跳转。

跳出双层for循环的两种方法：（一共四种，其余两种即利用break或continue，设立标签label来进行。这里不作展示）

	//跳出两层for循环的方法——展示goto用法
	//第一种：设置全局标志位
	flag := false
	for i := 0; i < 5; i++ {
		for j := 0; j < 5; j++ {
			if i == 2 && j == 2 {
				flag = true
				break
			}
			fmt.Printf("i=%d--j=%d\n", i, j)
		}
		if flag {
			break
		}
	}
	fmt.Println()

	//第二种：使用goto
	for i := 0; i < 5; i++ {
		for j := 0; j < 5; j++ {
			if i == 2 && j == 2 {
				goto breakTag
			}
			fmt.Printf("i=%d--j=%d\n", i, j)
		}
	}

breakTag: //所定义label
	fmt.Println("跳出两层for循环")

goto尽量少用，代码较多时容易逻辑混乱，造成代码阅读障碍。
